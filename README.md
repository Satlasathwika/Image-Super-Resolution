Image Super Resolution using ANN

Overview

This project enhances low-resolution images using an Artificial Neural Network (ANN). The model is trained to convert blurred or low-quality images into clearer and higher-resolution outputs.
It helps improve image quality by reconstructing details from low-resolution inputs.

Features

Upload low-resolution images
Enhance image quality using ANN
Simple and user-friendly interface
Real-time image processing
High-resolution image generation

Tech Stack

Python
TensorFlow / Keras
NumPy
OpenCV
PIL (Python Imaging Library)

How It Works

Upload a low-resolution image
Image gets preprocessed
ANN model predicts a higher-resolution image
Enhanced image is displayed

Applications

Medical imaging
Satellite image enhancement
Old photo restoration
Security and surveillance
Digital media improvement
